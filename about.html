About SF Crime Analytics

Our Mission

SF Crime Analytics is dedicated to providing transparent, accurate, and accessible information about crime patterns in San Francisco. Our goal is to help residents, businesses, and law enforcement make data-driven decisions about public safety.

Data Source

We use the official San Francisco Police Department (SFPD) Incident Reports dataset, available through the San Francisco Open Data Portal. This dataset is regularly updated and contains detailed information about reported incidents across the city.

API Endpoint: https://data.sfgov.org/resource/wg3w-h783.json

The data includes:

  • Incident dates and times
  • Precise locations
  • Incident categories and descriptions
  • Police districts
  • Resolution status

Technology Stack

Our platform is built using modern web technologies:

  • Leaflet.js for interactive maps
  • Chart.js for data visualization
  • Custom CSS for responsive design
  • Vanilla JavaScript for data processing

Data Updates

The SFPD dataset is updated regularly, and our visualizations reflect the most current available data. We process and display this information in real-time to ensure accuracy and relevance.
